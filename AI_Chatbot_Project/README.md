# Week 1 - AI Chatbot Using Python

## Introduction
This project involves developing a simple AI Chatbot using Python. The chatbot interacts with users by responding to predefined questions and provides basic conversational functionality.

## Project Objectives
- Understand the fundamentals of AI chatbots.
- Implement a rule-based chatbot using Python.
- Learn user input handling and conditional statements.
- Gain practical experience in Python programming.

## Technologies Used
- Python
- VS Code
- GitHub

## Project Workflow
1. Design chatbot conversation flow.
2. Create predefined questions and responses.
3. Implement chatbot logic using Python.
4. Test the chatbot with different inputs.
5. Improve responses and functionality.

## Implementation Steps
1. Install Python and VS Code.
2. Create a Python file (`chatbot.py`).
3. Write chatbot code using input and conditional statements.
4. Run and test the chatbot.
5. Upload the project to GitHub.

## Features
- Interactive text-based conversation.
- Responds to common user questions.
- Simple and easy-to-understand implementation.
- Beginner-friendly AI project.

## Results
The chatbot successfully accepts user input and provides predefined responses, demonstrating basic conversational AI concepts.

## Learning Outcomes
- Learned Python fundamentals.
- Understood chatbot development concepts.
- Improved problem-solving and programming skills.
- Gained experience with GitHub project management.
