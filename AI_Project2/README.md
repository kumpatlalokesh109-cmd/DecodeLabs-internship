# Week 2 - Data Classification Using AI

## Introduction
This project focuses on building a machine learning classification model using the Iris Dataset. The model is trained to classify flowers into different species based on their features.

## Project Objectives
- Understand the basics of machine learning classification.
- Learn how to work with datasets.
- Implement the K-Nearest Neighbors (KNN) algorithm.
- Evaluate model performance using classification metrics.

## Technologies Used
- Python
- Jupyter Notebook / VS Code
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Dataset
The Iris Dataset contains measurements of iris flowers, including:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target Classes:
- Setosa
- Versicolor
- Virginica

## Project Workflow
1. Load the Iris Dataset.
2. Preprocess the data.
3. Split data into training and testing sets.
4. Train the KNN classification model.
5. Make predictions.
6. Evaluate performance using Confusion Matrix and F1 Score.

## Implementation Steps
1. Import required libraries.
